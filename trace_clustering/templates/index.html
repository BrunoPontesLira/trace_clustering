Descrever sobre a aplicação
Em grandes empresas, vários são processos executados a todo momento. Como muitos desses processos são executados com ajuda dos sistemas de TI, eles deixam “pegadas digitais”, o que chamamos de log de eventos (Dumas et al., 2018). Considerando um processo em particular, define-se como trace uma sequência de execuções das atividades relacionadas ao processo. Cada uma dessas execuções é dita ser um evento, o qual pode ser descrito em termos de atividade executada, hora e data de execução, recursos envolvidos, entre outros. Uma coleção de traces compõe um log de eventos, no qual são registradas, portanto, as execuções de um processo. São geradas grandes quantidades de dados que, se analisadas rapidamente, podem gerar valor significativo para a empresa.
